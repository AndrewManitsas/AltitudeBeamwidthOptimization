# Altitude Beamwidth Optimization

**Author:** Andreas Manitsas  
**Course:** UAV13 Topics to Wireless Communications (MSc Aerial Autonomous Systems)  
**Institution:** Aristotle University of Thessaloniki - Department of Electrical & Computer Engineering

This repository contains the simulation code and the LaTeX report for the semester assignment of the **UAV 13: Advanced Topics to Wireless Communication** course.

The project evaluates the performance trade-offs associated with the joint optimization of UAV flying altitude (H) and directional antenna beamwidth (Theta). It implements a fly-hover-and-communicate protocol across a hexagonally tessellated ground area, analyzing three fundamental multiuser communication models:
* Downlink Multicasting
* Downlink Broadcasting
* Uplink Multiple Access (MAC)

The simulation models reproduce the mathematical frameworks and numerical results from the reference paper: *Joint Altitude and Beamwidth Optimization for UAV-Enabled Multiuser Communications* (He et al., 2018).

## Repository Structure
* `code/main.py`: Core Python simulation script. Generates the system performance metrics and plots for the three communication models.
* `report/report.tex`: The LaTeX source file for the final assignment report.
* `report/report.pdf`: The final assignment report.
* `report/Figure_1.png`: The output plots generated by the simulation, formatted for direct inclusion in the LaTeX document.

## Dependencies
The simulation is built using standard Python scientific computing libraries. Ensure you have the following installed:
* Python 3.8+
* NumPy
* Matplotlib
